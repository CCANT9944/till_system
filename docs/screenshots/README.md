# Screenshots

This folder is the intended home for public screenshots used by the repository landing page.

Suggested image names:

- `till-overview.png` — main till grid with category buttons and cart
- `product-details.png` — Product Details tab with search and table view
- `bills-tab.png` — Bills tab with history and totals
- `manager-dialog.png` — Manager dialog showing Product and Design actions

Suggested capture rules:

- Use realistic but non-sensitive sample data only
- Avoid showing private local paths, real customer information, or live PIN values
- Prefer desktop-sized captures with the full main window visible
- Keep file sizes reasonable so the GitHub page stays fast
